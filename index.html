Nombre de la institucion

Breve historia: Fundada en 1975 por Bill Gates y Paul Allen, Microsoft creció desde sistemas operativos y software de productividad hacia servicios de nube, plataformas de inteligencia artificial y hardware. En su historia reciente, la empresa se ha orientado fuertemente a la nube mediante Azure y a la integración de IA en todos sus productos, cambios estratégicos reflejados en sus informes corporativos.

Visión

Visión (sintetizada)

Microsoft busca liderar la transformación digital global mediante plataformas y servicios que permitan a personas y organizaciones alcanzar más objetivos, impulsando innovación sostenible y accesible. (Paráfrasis de su enfoque estratégico y metas corporativas), en estos años Microsoft en el 2024 nos dijo que la visión actual es “democratizar la IA, haciéndola accesible y beneficiosa para todos”.

Misión

Misión (oficial y paráfrasis)

Misión oficial (resumida): “Empoderar a cada persona y organización del planeta para lograr más.”

Paráfrasis: Microsoft desarrolla tecnología, plataformas y servicios que facilitan la productividad, colaboración y toma de decisiones, buscando que tanto usuarios individuales como empresas puedan incrementar su capacidad operativa y de innovación.

Cantidad de información

Cantidad de información que maneja la organización

Microsoft administra una cantidad de información extremadamente elevada debido a la naturaleza global de sus operaciones y a los servicios que ofrece. En primer lugar, maneja telemetría generada por sus propios productos, como Windows, Office, Edge y Xbox. Estos datos incluyen registros de uso, métricas de rendimiento, actualizaciones, fallos y eventos que permiten mejorar las características y la estabilidad del software. Además, procesa datos empresariales de clientes que utilizan la nube de Azure y Microsoft 365. Allí se almacenan correos electrónicos, documentos, bases de datos, aplicaciones corporativas, información operativa y procesos de negocio de miles de organizaciones. A esto se suman los datos internos relacionados con la administración de sus plataformas globales, como monitoreo de centros de datos, flujos de red y análisis de seguridad. El volumen total es masivo y se incrementa constantemente. Esta realidad exige sistemas avanzados para clasificación, almacenamiento, protección, cumplimiento legal y políticas estrictas de retención. Microsoft debe ajustarse a diversas normativas de privacidad según cada país y región, lo que añade complejidad operativa. En síntesis, Microsoft gestiona información a escala planetaria, lo cual requiere infraestructura robusta, procesos maduros y tecnologías capaces de operar de forma continua y segura. Tomemos en ejemplo de almacenamiento de la nube que tiene 1 TB de almacenamiento para uso por usuario y 6 TB de almacenamiento en plan familiar, tomemos en cuenta la cantidad de información que tienen Microsoft que es una empresa global de millones y millones de personas que usan sus servicios

Cómo almacena la información

El almacenamiento de información en Microsoft se basa principalmente en su plataforma en la nube Microsoft Azure, OneDrive entre otros combinando arquitectura de nube pública, privada e híbrida. Esto le permite ofrecer flexibilidad a los clientes y mantener redundancia en diferentes regiones del mundo. La empresa utiliza servicios de almacenamiento escalables, como Azure Blob Storage, Azure Files y Data Lake Storage, que permiten almacenar desde archivos simples hasta grandes volúmenes de datos analíticos. Además, emplea bases de datos administradas como Azure SQL Database, Cosmos DB y PostgreSQL/ MySQL gestionados, los cuales permiten un manejo eficiente de datos estructurados y no estructurados. También dispone de modelos híbridos mediante Azure Arc, con los que las organizaciones pueden conectar su infraestructura local con la nube, integrando políticas de seguridad, administración y monitoreo de forma centralizada. En resumen, Microsoft almacena la información combinando diferentes tipos de servicios para garantizar confiabilidad, velocidad de acceso, disponibilidad continua y seguridad reforzada.

Tipo de hardware

Microsoft opera una red mundial de centros de datos de gran escala, considerados entre los más avanzados de la industria. Estos centros de datos están equipados con:

• Servidores de alto rendimiento, optimizados para cargas intensivas de cómputo, virtualización de máquinas y uso de IA.

• Unidades de procesamiento especializadas, como GPUs avanzadas y chips diseñados específicamente para entrenamiento e inferencia de inteligencia artificial.

• Sistemas de almacenamiento distribuido, conformados por miles de unidades SSD y discos de alta velocidad que garantizan redundancia, seguridad y disponibilidad.

• Infraestructura de red de alta capacidad, con switches y routers diseñados para manejar tráfico a escala global, conectando centros de datos entre continentes con baja latencia.

• Racks modulares y sistemas de enfriamiento eficientes, que permiten optimizar el uso energético y mantener la continuidad del servicio.

Tecnología de comunicaciones

La infraestructura de comunicaciones de Microsoft se basa en una red global privada, diseñada para conectar sus centros de datos y garantizar el intercambio seguro y rápido de información. Entre las principales tecnologías utilizadas destacan:

• Redes de fibra óptica de alta velocidad, propias y en alianza con otros proveedores globales.

• Sistemas de interconexión (peering) con múltiples operadores alrededor del mundo para reducir la latencia en conexiones de usuarios y empresas.

• VPN y ExpressRoute, que permiten a los clientes conectar sus oficinas o centros de datos directamente con Azure a través de enlaces privados, sin pasar por Internet público.

• CDN (Content Delivery Network) para distribuir contenido de forma rápida y estable a usuarios en cualquier región del mundo.

• Redes definidas por software (SDN) que permiten configurar, monitorear y administrar las comunicaciones de forma inteligente y automatizada.

Sistemas de información

• Microsoft 365 / Office 365: plataformas para productividad empresarial que incluyen correo, almacenamiento en la nube, herramientas colaborativas y aplicaciones de oficina. Su objetivo es mejorar el trabajo en equipo y la gestión documental.

• Dynamics 365: conjunto de soluciones ERP y CRM que ayudan a las empresas a administrar finanzas, ventas, operaciones y atención al cliente.

• Sistemas internos de gestión: usados para administrar recursos humanos, contabilidad, seguridad, cumplimiento y coordinación interna.

Sistemas de Base de datos

• Azure SQL Database: base de datos relacional administrada, ideal para sistemas transaccionales.

• Cosmos DB: base de datos NoSQL distribuida globalmente, diseñada para aplicaciones que requieren baja latencia.

• Azure Data Lake: plataforma para almacenar y procesar grandes volúmenes de datos analíticos.

Sistemas de comunicaciones

• Teams, Exchange Online, servicios de voz y colaboración: facilitan comunicación interna, soporte y atención a clientes.

• Redes globales de Azure (ExpressRoute, VPN, CDN): para interconexión empresarial y distribución de contenido.

Sistemas de procesos

• Azure DevOps y GitHub: plataformas que permiten gestionar proyectos, control de versiones, integración continua y despliegue de software.

• Azure Monitor y Microsoft Sentinel: sistemas de observabilidad y seguridad que permiten detectar incidentes, analizar registros y prevenir amenazas.
